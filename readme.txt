Embed a Google Map by address. No API key required.

== Description ==

A lightweight Gutenberg block that embeds a Google Map using an iframe. Just type an address and you're done — no API key, no billing, no configuration.

Features:

* Address-based location search
* Adjustable zoom level (1–21)
* Configurable map height
* Roadmap and satellite views
* Wide and full-width alignment support

== Installation ==

This block is loaded as a must-use plugin. Place the `flashblocks-map` directory in `wp-content/mu-plugins/` and require it from `mu-plugins/index.php`.
